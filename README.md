# depressor

Toy text-compression utility using Huffman Coding.

## Usage
```sh
$ depressor -help
usage:
        depressor -c file.txt # outputs file.txt.mkti
        depressor -d file.txt.mkti # outputs file.txt
        depressor -help
```
